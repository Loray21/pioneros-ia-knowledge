# Ejercicio

``` txt
Aprender a aprovechar la IA para transformar reuniones (internas o con clientes) en información útil y accionable, reduciendo el tiempo dedicado a escribir reportes o preparar presentaciones.

🪄 Paso 1: Transcribí una reunión real
Podés grabar una reunión de equipo, una conversación con un cliente o incluso una clase.
Luego, generá la transcripción con alguna de estas herramientas:

Tactiq (plugin gratuito para Google Meet o Zoom)

ChatGPT o Lindy

Alternativas: Otter.ai, Notta, o Fireflies.ai

🧩 Paso 2: Extraé los puntos clave
Usá ChatGPT para identificar los temas principales, decisiones tomadas y tareas pendientes.
Podés darle un prompt como este:

“Resumí los puntos más importantes de esta reunión en un listado con decisiones, acciones y responsables.”

🎨 Paso 3: Generá una presentación
Usando los puntos clave o la transcripción, creá una presentación con Gamma.app.

💪 Resultado esperado:
Una presentación lista para compartir (por ejemplo, en Gamma o PowerPoint).

Un resumen de decisiones y próximos pasos elaborado con IA.

```
